# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-22

### Added
- NUI vehicle spawner for 124 GabZ vehicles across 13 categories
- Search by name, brand, and spawn code
- Multi-language support (German & English) via JSON locale files
- Configurable spawn options (delete previous vehicle, warp, upgrades)
- Optional ace permission system
- Server-side spawn code validation
