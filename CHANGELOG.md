# Changelog

This file describes the changes in the latest release of gabz-vehicle-spawner. For older versions, see [GitHub Releases](https://github.com/MrPresidentWhite/gabz-vehicle-spawner/releases).

## 1.0.0

### Added
- Vehicle spawner menu for 124 GabZ vehicles across 13 categories (Super, Sports, Emergency, and more)
- Search by vehicle name, brand, or spawn code
- Category filters in the sidebar for quicker browsing
- German and English language support for the menu and in-game notifications
- Open the menu with F7 or the `/gabzvehicles` chat command
- Option to delete your previous spawned vehicle when selecting a new one
- Option to warp directly into the spawned vehicle
- Optional admin permission check to restrict who can open the menu and spawn vehicles
- Server-side validation so only allowed GabZ spawn codes can be used

### Changed
- Menu panel stays readable while the game world remains visible around it
